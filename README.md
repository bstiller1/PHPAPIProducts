# PHP API CRUD APP

## Starting point:

`public/index.php`

## CRUD Operations

1. Create Product: `create.php`
2. Read Products: `read.php`
3. Update Product: `update.php`
4. Delete Product: `delete.php`

**View**: `/mvc/views/user_profile.php`<br>
**Controller**: `/mvc/controllers/UserController.php`
